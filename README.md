# Tingle
"tingle"은 "기분 좋은 소름"이라는 뜻으로 귀를 즐겁게 해주는 음악 스트리밍 서비스입니다.

## 사용기술
Frontend
- react
- react-router
- redux-toolkit
- styled-components
- MUI

Backend
- Node.js
- MongoDB
